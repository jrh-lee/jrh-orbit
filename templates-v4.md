# JRH-Orbit 템플릿 v4 — SPEC.md §2 교체용

> 공통 양식: 헤더(`#`/`##`/`###`) → 바로 불릿(`-`) → 빈 줄 2개 → 다음 섹션
> 체크박스(`- [ ]`) 섹션은 예외
> 각 섹션에 유도 질문을 주석으로 포함 → 뭘 적어야 할지 바로 보임

---

## 공통 서식 규칙

```
## 섹션 제목
- 내용 1
- 내용 2


## 다음 섹션
- 내용 3


### 서브 섹션
- 내용 4
```

- 모든 헤더(`#`, `##`, `###`) 바로 아래에 빈 줄 없이 내용 시작
- 내용은 불릿(`-`)으로 시작
- 불릿 내에서 줄바꿈이 필요하면 같은 불릿 아래 들여쓰기
- 섹션과 섹션 사이 빈 줄 **2개**
- **예외**: 체크박스(`- [ ]`), 테이블(`| |`)은 불릿 대신 해당 형식 사용

## 7. Quick Memo 승격 시 매핑 규칙

승격할 때 기존 Quick Memo 본문이 어디로 가는지:

| 승격 대상     | 메모 본문 → 이동 위치          | 자동 추가되는 스켈레톤                       |
| ------------- | ------------------------------ | -------------------------------------------- |
| Analysis Note | → `## 목적` 의 불릿으로        | 분석 조건, 결과, 결론, 코드 참조, 후속 과제  |
| Design Note   | → `## 배경 / 문제` 의 불릿으로 | 의사결정 요약, 대안, 선정 근거, 리스크, 결론 |
| Study Note    | → `## 핵심 내용` 의 불릿으로   | 주제, 출처, 적용 가능성, 추가 조사, 요약     |
